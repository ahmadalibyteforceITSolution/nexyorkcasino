# NEXYORK 🗽 - The Premium NY Super-App

NexYork is a high-engagement, gamified platform designed for the modern New York lifestyle. It combines city discovery, live sports tracking, and a premium casino experience in one unified ecosystem.

## 🚀 Key Features

- **Live Sports Tracker**: Real-time match scores and odds for all major NY teams (NBA, MLB, MLS) powered by Socket.io.
- **Global Live Casino**: Real-time roulette feed and betting terminal with synchronized data across the world.
- **City Hotspots**: Curated guides to the best dining, viewpoints, and nightlife in New York.
- **Premium Aesthetics**: Dark-mode, glassmorphism design built with Vue.js 3 and Tailwind CSS.
- **Real-Time Database**: MongoDB integration for dynamic data updates.

## 🛠 Tech Stack

- **Frontend**: Vue.js 3, Tailwind CSS, Vite
- **Real-time**: Socket.io
- **Backend**: Node.js, Express
- **Database**: MongoDB (Mongoose)
- **Deployment**: Optimized for Vercel

## 📦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Everything (Frontend + Backend)**:
   ```bash
   npm run dev
   ```

## 🌐 Deployment on Vercel

To deploy on Vercel:
1. Connect your GitHub repository to Vercel.
2. Set the **Build Command** to `npm run build`.
3. Set the **Output Directory** to `dist`.
4. Add your environment variables (like `MONGODB_URI`) in the Vercel dashboard.
